# 🎬 YeaMovie

YeaMovie — учебное веб-приложение для поиска и просмотра информации о фильмах и сериалах. Проект разработан на чистом JavaScript с использованием внешнего API.

## ✨ Функционал

* Получение данных с API
* Поиск фильмов по названию
* Фильтрация по году, жанру, стране и возрастному рейтингу
* Просмотр популярных фильмов, сериалов и подборки фильмов
* Модальное окно с подробной информацией о фильме
* Обработка ошибок при загрузке данных
* Адаптивная верстка

## 🛠️ Используемые технологии

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API
* DOM API
* Flexbox
* CSS Grid
